# Distributed Appointment Database System

Government appointment booking system using custom data structures in C++.

## Data Structures

- **B-Tree**: Sorted storage, O(log n) operations
- **Hash Table**: O(1) CNIC lookup
- **API Layer**: Manages data structure operations

## Tech Stack

- Backend: C++17
- Frontend: HTML/CSS/JavaScript
- Deployment: AWS EC2 (planned)

## Structure

src/ # C++ backend
include/ # Header files
web/ # Frontend pages
